# Housebook

> Aplicacao de listagem de imoveis desenvolvida em HTML, CSS e JavaScript puro, com busca e filtragem de propriedades.

## Stack

- HTML5
- CSS3
- JavaScript

## Prerequisitos

- Nenhum prerequisito especifico

## Como executar

```bash
# Clone o repositorio
git clone https://github.com/meirelesdev/housebook.git
cd housebook
```
